>Describe one difference between SQL and other programming languages.
SQL is declarative rather than a procedural language.

>In your own words, explain how the pieces of a database system fit together at a high level.
Databases display data using a table structure that define what the data represents.  The way it works is databases allow you 
to query and change the data.  Then we can run SQL to get results, update or change the way we store the data.

>Explain the meaning of table, row, column, and value.
 Columns contain the name of the information stored in the cell for that column. Rows contain the data for the columns, and the 
 value is what is found in the cell
